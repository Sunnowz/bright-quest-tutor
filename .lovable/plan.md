

## ЭдуТьютор — ИИ-платформа для учеников

### Общий подход
Веб-приложение с тремя основными модулями, авторизацией через Lovable Cloud и ИИ через Lovable AI Gateway. Интерфейс на русском языке, адаптивный дизайн для всех возрастов.

---

### 1. Авторизация и профиль ученика
- Регистрация/вход по email + пароль
- Профиль ученика: имя, возраст, класс, список интересов/хобби (Minecraft, космос, футбол и т.д.)
- Выбор предметов, которые изучает ученик
- Таблица `profiles` с полями: имя, возраст, класс, интересы (массив), предметы

### 2. ИИ-Тьютор (чат-бот с методом Сократа)
- Чат-интерфейс с выбором предмета и темы
- Streaming-ответы через edge function + Lovable AI
- Системный промпт: «Никогда не давай готовый ответ. Задавай наводящие вопросы. Используй метод Сократа. Хвали за правильный ход мысли.»
- Образовательные фильтры: ИИ отклоняет вопросы не по теме
- История чатов сохраняется в базе

### 3. Умный конспект
- Загрузка аудиофайла урока
- Транскрипция через ElevenLabs Speech-to-Text (коннектор)
- ИИ обрабатывает транскрипцию → структурированный конспект с выделенными терминами
- Автогенерация проверочного теста (5-10 вопросов) по материалу конспекта
- Ученик проходит тест прямо в приложении, видит результат

### 4. Геймификация задач
- Ученик выбирает предмет и тему
- ИИ генерирует задачу, адаптируя условие под интересы из профиля (например, задача про Minecraft вместо поездов)
- Пошаговая проверка решения с подсказками
- Система очков/достижений за решённые задачи

### 5. Навигация и дизайн
- Боковое меню: Тьютор, Конспекты, Задачи, Профиль
- Яркий, дружелюбный дизайн с фиолетово-синей палитрой
- Адаптивный интерфейс (мобильный + десктоп)
- Дашборд с прогрессом: количество решённых задач, пройденных тестов, часов с тьютором

### Технический стек
- **Frontend**: React + Tailwind + shadcn/ui
- **Backend**: Lovable Cloud (Supabase) — авторизация, база данных, edge functions
- **ИИ**: Lovable AI Gateway (gemini-3-flash-preview) для тьютора, конспектов и генерации задач
- **Транскрипция**: ElevenLabs STT через коннектор
